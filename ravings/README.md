Ravings
-------
С сервера идёт последовательность символов неопределённой длины, периодически появляется код некой функции QCTF(flag) , которая генерирует последовательность символов по флагу. Можно заметить, что функция печатает символы флага как есть и случайные символы, которые во флаге быть не могут.

для запуска server.py нужен Tornado (асинхронный веб фреймворк)
http://www.tornadoweb.org
